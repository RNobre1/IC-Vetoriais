# Índice CPU-only do PyTorch.
# Motivo: ADR vault/decisões/2026-04-28-modelo-embedding-minilm.md fixa MiniLM-L6-v2
# em CPU. Sem este `--extra-index-url`, o pip resolve `torch` para a build CUDA

# vault/lições/2026-05-10-torch-cpu-only-vs-cuda.md.
--extra-index-url https://download.pytorch.org/whl/cpu
